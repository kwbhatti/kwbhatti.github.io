Reports

Select the link in the top navigation for Summary (Summary of the full report), Full Report (Details report for each row validation), Split (Split for each indicator), or Failures (All the failures for all indicators)

  • Summary: Outline report for the Data Validation Framework including total number for each indicator and all indicators combined
  • Full Report: Detailed report for the full execution for all indicators, all failures are highlighted in Residential
  • Split Report: Individual report for each indicator, click on the indicator to view specific report on page
  • Failures: All the data sets which failed the validation during the execution
